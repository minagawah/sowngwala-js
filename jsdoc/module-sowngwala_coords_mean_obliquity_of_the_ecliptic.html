Methods

(static) mean_obliquity_of_the_ecliptic(dt) → {number}

The method is used in 'equatorial_from_ecliptic_with_generic_date' when it tries to find "obliquity of the ecliptic (ε)" which is the angle between the planes of the equator and the ecliptic. (Peter Duffett-Smith, p.41) The Rust version only take "date" but we want to make it accurate so that it now takes "time" for this version.
